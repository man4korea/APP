🧪 테스트 시나리오

1. 회사 등록 테스트

새 회사를 등록하고 설립자 계정을 생성하는 전체 플로우를 테스트하세요.

2. 지점 등록 테스트

본점에서 지점을 등록하고 지점 관리자를 지정하는 기능을 테스트하세요.

3. 지점 통합 테스트

독립 회사가 본점의 지점으로 통합되는 승인 프로세스를 테스트하세요.

4. 권한 관리 테스트

다양한 역할의 사용자가 적절한 권한으로 시스템에 접근하는지 확인하세요.
